ES6 Higher Order Array Methods

  • 1. forEach() : Iterates over the array using a callback fucntion.
  • 2. filter() : returns an array using a callback function that satisfies a set boolean criteria.
  • 3. map() : manipulate values of the array using a callback function and returns a new array.
  • 4. shift(): removes first element and changes the configuration of the original array.
  • 5. slice(): returns an array as a new one.
  • 6. sort() : sorts the array.
  • 8. find() : finds and returns the element that fits the boolean criteria you are looking for.
  • 9. reduce() : reduces the elements into a single entity like and accumulator.
